# Popover
Popover is a non-modal dialog that floats around a trigger.

## Import
```
import { Popover } from "@deca-ui/react"
```

## Usage
Note: The Popover.Trigger component only accepts one child component and automatically passes `onClick` to it.
<ComponentWrapper>
  <Popover>
    <Popover.Trigger>
      <Button>Open Popover</Button>
    </Popover.Trigger>
    <Popover.Content>This is the content of the popover.</Popover.Content>
  </Popover>
</ComponentWrapper>
```
<Popover>
  <Popover.Trigger>

```

export default ({ children }) => <Layout mdx>{children}</Layout>
